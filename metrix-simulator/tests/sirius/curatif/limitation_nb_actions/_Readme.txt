Test de limitation du nombre d'actions curatives
----------------------------------
Il s'agit du même test que l'étude vendée avec curatif parades + TD.
Pour la variante 3, le curatif optimal est d'ouvrir 3 DJ et de bouger le TD en même temps
Ce qui n'est pas possible puisque le nombre d'action est limité à 3.
