Tests de defaut groupe
----------------------
Ce test produit les mêmes résultats que HADES en actif-seul avec la compensation proportionnelle à Pmax
Variante 0 : Le groupe en défaut 'FSSV.O11_G' produit 480 MW, compensés par les autres groupes
Variante 1 : Le groupe en défaut ne produit pas, pas de modification des flux
Variante 2 : Le groupe en défaut est déconnecté, pas de modification des flux
